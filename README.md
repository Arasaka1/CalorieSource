# CalorieSource

SideStore source repository for the **Calorie** app — an AI-powered calorie tracker for iOS 26.

## Add to SideStore

Open SideStore, go to **Sources** → tap **+** → paste this URL:

```
https://arasaka1.github.io/CalorieSource/apps.json
```

## Features

- Apple Liquid Glass UI design
- AI-powered nutritional analysis (Gemini, Cerebras, Apple Foundation Models)
- Activity rings and weekly progress tracking
- On-device backup and restore
